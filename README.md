## 🧡 kiosk
#### 키오스크 관리 프로젝트 (팀 프로젝트)
1. 키오스크 개인 프로젝트 디벨롭 과제
2. 고객은 기존에 만든 키오스크 프로그램을 통해 메뉴와 상품을 조회해 주문을 하고 대기번호를 받음
3. 상점은 키오스크 관리 프로그램을 통해 메뉴와 상품 생성/삭제하고, 대기/완료 주문 목록을 관리
